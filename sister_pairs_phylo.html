Sister pairs selection for syntopy analysis

Author

Matěj Tvarůžka

In this code I will select sister species pairs of North American passerines for the subsequent analysis of syntopy.

Firstly we are going to select the phylogeny of wood-warblers to try the workflow. We can get the acces to eBird taxonomy files using taxonomyGet().

With the taxonomy loaded, we select the family of wood-warblers (Parulidae).
